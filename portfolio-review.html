Methodology

Time-weighted return is calculated by linking daily sub-period portfolio returns while neutralizing external deposits and withdrawals. This is generally the preferred measure for evaluating investment management performance because it reduces the distortion caused by when a client adds or removes money.

Money-weighted return / IRR is useful for understanding the investor's actual dollar experience. This page prioritizes TWR for benchmark comparison and includes benchmark-relative statistics such as beta, alpha, tracking error, information ratio, and capture ratios.

Data note: this static GitHub Pages version uses public browser-side market data endpoints. For compliance-grade client reporting, connect the same calculation engine to a controlled backend and licensed data source.

Educational analytics only. Not investment advice. Verify holdings, transaction classification, prices, corporate actions, and benchmark selection before using in a client report.
